# GoodyOS Package Rationale

Why each package list exists and what it provides. See **README.md** and **GoodyOS_Project_Bible.pdf** for full stack.

| List | Purpose |
|------|---------|
| **base** | Minimal Debian core: kernel, network, init, nftables, no desktop |
| **desktop** | KDE Plasma, Latte Dock, Konsole, Dolphin — daily driver DE |
| **security** | Nmap, Wireshark, aircrack-ng, macchanger, UFW, Mullvad VPN |
| **dev** | build-essential, Git, Docker, Python 3 + GI for Ghost |
| **ai-dev** | Ollama, Jupyter, Python ML stack; optional CUDA |
| **forensics** | nwipe (DoD/NIST), Autopsy, Testdisk, Foremost |
| **productivity** | LibreOffice, VLC, Thunderbird, GIMP, Inkscape, Kdenlive, Firefox ESR |

All choices follow **Zero Bloat** and **Proven Trust** from the Project Bible.
